# Next.js Starter — Demo Repo

A minimal Next.js app used to demo FileMind's codebase navigation.

## Structure

- `src/utils/jwt.ts` — JWT sign/verify utilities
- `src/middleware/auth.ts` — Auth middleware using JWT
- `src/pages/api/login.ts` — Login endpoint

## Try it with FileMind

```
npx tsx agent/src/index.ts "How does authentication work?" ./demo/sample-repos/nextjs-starter
```
